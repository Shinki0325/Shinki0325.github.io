  - label: 它首先是一种“闭合网络”，不是后来那种一开浏览器就到处互通的互联网
    original: >-
      パソコン通信はいわゆる「クローズドネットワーク」であり、特定のサーバ（ホスト）とその参加者（会員）の間だけをつなぐ閉じたネットワークであったため、他のネットワークに接続するには一度接続を切る必要があった。
    translation: >-
      条目一开始就点出了最重要的差别：PC 通信本质上是一个封闭网络。你连上的是某个具体服务和它的会员共同体，而不是像后来的互联网那样，可以在同一次连接里自然跳到整个开放网络的其他角落。
    note: 这段很适合说明九十年代玩家共同体为什么常常是分论坛、分服务、分入口地长出来的。
    focus: true
  - label: 用户靠电话线拨号接入，在里面收邮件、泡论坛、聊天、传文件
    original: >-
      パーソナルコンピュータ（パソコン）に音響カプラ、後にはモデムを接続して一般加入回線（電話回線）を経由してサーバにダイアルアップ接続していた。その中で電子メールの送受信や電子掲示板、チャットなどを利用した情報交換が行われた。
    translation: >-
      在使用方式上，它是把电脑通过音响耦合器或调制解调器接上普通电话线，再拨号进入服务端；进去之后做的事情却已经很像后来熟悉的网络生活了，包括收发电子邮件、逛电子公告板、聊天室交流和文件交换。
    note: 这块很适合给观众建立“网页前夜并不是没有网络社交”的直观印象。
    focus: true
  - label: 全盛期正好横跨八十年代后半到九十年代，然后被互联网普及逐步替代
    original: >-
      全盛期は1980年代後半から1990年代で、のちにインターネットが一般ユーザーに開放されたため、徐々に衰退していった。
    translation: >-
      从时间上看，PC 通信的黄金时期恰好就是八十年代后半到九十年代。也正因为这样，它和你现在整理的那批九十年代游戏、动画、玩家社群几乎是同步扩张的，直到互联网向普通用户开放之后才慢慢退场。
    note: 这句能直接把网络基础设施和你的视频主时间线扣在一起。
    focus: true
  - label: 大型服务会把同好组织成“论坛”，这就是很多玩家共同语言的温床
    original: >-
      大規模なところでは、「コミュニティー」と呼ばれる趣味・話題を共通にする集まり（ニフティサーブではフォーラム、PC-VANではSIGと呼んだ）をいくつも作り、それぞれの中で情報交換ができるようにしていた。
    translation: >-
      规模较大的服务不会只提供一个总入口，而是把不同兴趣的人拆进各自的社区里。NIFTY-Serve 叫“论坛”，PC-VAN 叫 SIG，但本质都是围绕某种话题聚拢稳定成员，让信息交换、口碑扩散和共同语言在里面发酵。
    note: 这段可以直接拿来解释“为什么很多作品会先在特定圈层里迅速形成共识，再往外扩散”。
