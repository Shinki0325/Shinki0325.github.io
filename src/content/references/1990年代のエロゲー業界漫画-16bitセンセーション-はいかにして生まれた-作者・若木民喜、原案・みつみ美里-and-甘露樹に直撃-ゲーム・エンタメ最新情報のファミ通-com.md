  - label: 这部漫画最初就是从饭桌上的聊天长出来的
    original: >-
      3人で会食をすると、若木先生が描いてる漫画のネタについてよくお話するんです。あるとき、その話のなかで『16bitセンセーション』のもとになるような話が出たんだと思います。でも、その後私はその話をすっかり忘れてしまい、しばらく経ってから若木先生から「みつみさん、アレやりませんか？」と声を掛けてもらいました。
    translation: >-
      みつみ美里回忆，他们三个人一起吃饭时，经常会聊若木民喜正在画什么漫画点子。《16bit Sensation》的源头，大概就是某次饭桌上谈出来的一个念头；只是她后来自己都忘了，直到过了一阵子，若木又回来问她：“みつみさん，要不要来做那个？”
    note: 这段很适合说明它并不是编辑部命题作文，而是从圈内人私下共享的九十年代记忆里自然长出来的。
    focus: true
  - label: 它不是某个人的自传，而是很多人的碎片记忆拼出来的
    original: >-
      いろいろな人から「あのころこうだったよね」という話が集まって。なので、原案ではありますが、『16bitセンセーション』のストーリーは、私の自伝ではありません。漫画で描かれているネタは、基本的に若木先生が好きなゲームと、その作者との関わりを描いています。私の担当は、最初のきっかけと、小ネタ提供係です。私と甘露君の体験談は小ネタとして入っています（笑）。
    translation: >-
      这部漫画并不是みつみ或甘露树任何一人的自传。她明确说，很多“当年就是那样”的回忆都是从不同人那里慢慢汇拢来的；漫画里真正被画进去的，多半是若木喜欢的游戏和那些作品作者之间的关系，而她和甘露树更像是提供起点与细节轶事的人。
    note: 这块很适合拿来提醒读者，这类后见作品保存的是“集体记忆的编排”，不是逐条可核对的口述史。
    focus: true
  - label: 若木眼里，90 年代后半才是御宅文化真正开始发力的阶段
    original: >-
      1990年代後半からがオタクカルチャーの本番だと思うんです。いま出ている『16bitセンセーション』1巻は美少女ゲーム的にはまだ序章です。1990年代後半から2000年代の美少女ゲーム興隆期をがんばって描きます！
    translation: >-
      若木在访谈最后说得很直白：他认为御宅文化真正进入“本番”的阶段，其实是九十年代后半。已经出版的《16bit Sensation》第一卷，从美少女游戏史角度看还只是序章，真正想画的是从九十年代后半一路通向二十一世纪初的兴隆期。
    note: 这段和你现在这组“黄金年代”主线非常合拍，几乎可以直接嵌进主稿的时间判断里。
    focus: true
  - label: みつみ美里被回忆成改变整条画风河道的人
    original: >-
      絶対必要です！　あのね、エロゲー雑誌を見ていると、あるタイミングから突然同じ絵ばっかりになってくるんです。みつみ美里という存在が、エロゲーの画風を一度すべて塗り替えたんですよ。そういうところも、エロゲー業界のフットワークの軽さというか、すごいところですよね。
    translation: >-
      若木甚至直接说，在某个时间点之后翻黄油杂志，会突然觉得“怎么大家都画得差不多了”。在他的叙述里，みつみ美里这个名字几乎等于把当时整个黄油画风重新刷了一遍，这也正好说明那个业界跟风和吸收新风格的速度有多快。
    note: 这段很适合给主稿补“不是只有作品火，连审美模板都在快速重写”的层次。
