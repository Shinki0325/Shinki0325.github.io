    original: >-
      ９年前までテレ東系列が映らない地域に住んでいたので、アニメに関しては疎くてエヴァを観たのも２１世紀に入ってからでした。
    translation: >-
      这篇短回忆一开头就很能说明问题：作者直到九年前都还住在收不到东京电视台的地区，因此在动画经验上长期处于滞后状态，甚至连《EVA》都是到二十一世纪以后才补看的。
    note: 这句很适合和“地方不是同步追番共同体”那条主论点直接相连。
    focus: true
  - label: 搬到城市、能收到电视大阪之后，动画数量和设备投入会立刻一起升级
    original: >-
      それが結婚して都会に引っ越したらテレビ大阪が視聴できるようになり、アニメの放映数が増えてチェックのためにＨＤＤレコーダーの導入も速かったです。
    translation: >-
      作者后来搬到城市，能收到电视大阪之后，最直接的变化就是动画播出量明显增加，而为了跟得上这些节目，又很快导入了 HDD 录像机。也就是说，观看条件并不只是“多一个频道”，而是会连带改变设备选择和日常追看方式。
    note: 这一段能补足“传播条件一变化，用户行为会立刻重组”的现实层细节。
    focus: true
  - label: 就算后来又收不到，也能靠家人代录和录像租赁把观看链勉强接上
    original: >-

      しかし、バイクで数分の場所にあるレンタルビデオショップは尋常でない在庫数を誇る＆料金が安いので
    translation: >-
      但这种条件并不稳定。作者后来因为地面数字化，在单身赴任地又重新失去了相关频道，于是只能拜托妻子代录；与此同时，本地一家库存异常丰富、价格又低的录像租赁店，反而成了继续维持动画观看的重要补充渠道。
    note: 这块很适合说明观看环境其实是“频道 + 录像 + 代录 + 租赁”拼出来的一整套组合。
    focus: true
  - label: 旧作成批租、热门卷多备几套，说明租赁店曾是地方补番系统的一部分
    original: >-
      １週間かけて１期を観たので、２期をまとめてレンタル。人気作は１巻を２～３本入れるのでリリース当初以外は借りやすいです。
    translation: >-
      从作者描述的租赁习惯也能看出，当时地方补番并不是零散行为，而是有一套很成熟的替代路径：一周内看完第一季、再整批租第二季；热门作的第一卷甚至会多备两三套，只要错开新发售初期，基本都能借到。
    note: 这一段很适合补“地方补番系统是怎样实际运作的”，很生活，但很关键。
