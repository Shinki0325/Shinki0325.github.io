  - label: NIFTY-Serve 从 1987 年一路活到 2006 年，本身就是一整代网络史
    original: >-
      ニフティサーブ（NIFTY-Serve、のちにNIFTY SERVE）は、1987年から2006年までニフティ株式会社が運営していたパソコン通信サービス。
    translation: >-
      这条条目最先给出的，就是它惊人的存续长度。NIFTY-Serve 从 1987 年一直运营到 2006 年，几乎完整跨过了日本 PC 通信的繁荣期、互联网冲击期和最终退场期。
    note: 这句很适合把它当成“九十年代线上共同体”的代表性基础设施来介绍。
    focus: true
  - label: 它一开始就不是纯极客玩具，而是把论坛管理和社群运营做成制度化服务
    original: >-
      ニフティサーブはCompuServeからコマンド形態や画面様式、クレジットカード決済を使用したオンラインサインアップや会員規約の制定、フォーラムの管理をシステムオペレータに委任するという形態のコミュニティなどのノウハウをライセンスとして受けていた。
    translation: >-
      条目提到，NIFTY-Serve 不只是借了 CompuServe 的技术想法，而是把在线注册、会员规则、论坛管理和交由系统管理员运营社区这些做法一整套制度化地引进了日本。
    note: 这很适合说明九十年代玩家社群并不是“自然聊着聊着就有了”，而是建立在明确的运营结构之上的。
    focus: true
  - label: 到 1995 年它突破 100 万用户，已经是能决定信息流向的大平台
    original: >-
      1995年にはパソコンブームに乗じて会員数が100万人を突破。先行していたPC-VANの会員数を突破し、日本最多の利用者を擁するパソコン通信サービスとなった。
    translation: >-
      到 1995 年，NIFTY-Serve 的会员数已经突破 100 万，还反超了 PC-VAN，成了日本用户最多的 PC 通信服务。到了这个体量，它就不只是一个圈内角落，而是会真实影响哪些话题先发酵、哪些软件更容易扩散的平台。
    note: 这段很适合给“玩家共同语言为什么会集中形成”补足平台规模感。
    focus: true
  - label: 互联网来了之后，它不是一下死掉，而是先变得信息不够快、不够新
    original: >-
      専門的な内容を扱うフォーラムなど、あえてインターネットを利用する必要のないフォーラムは従来どおりの活動を続けるのだが、周囲の会員が減少することによってニフティサーブでは情報が不足するようになり、最新の情報を得るにはインターネットを併用する必要性が生じ始めた。
    translation: >-
      很有意思的一点是，它的衰退不是突然清空，而是先在信息速度上落后。条目写得很清楚：一些专业论坛本来还能继续活，但随着周边用户流失，光靠 NIFTY 内部已经拿不到最新消息，大家不得不同时去互联网补信息。
    note: 这块很适合拿来解释“旧共同体为什么会慢慢失去中心地位”，而不是简单说大家一夜之间迁移了。
