  - label: Sofmap 的起点其实是“做软件租赁”，不是一开始就卖成全国大连锁
    original: >-
      パソコンソフトのレンタル事業を目的として有限会社ソフマップを設立
      秋葉原1号店を開店
    translation: >-
      这条官方沿革最值得注意的，是 Sofmap 的起点并不是单纯卖电脑整机，而是以电脑软件租赁为目的成立，并且很早就在秋叶原开出第一家店。它从一开始就紧贴着软件流通和玩家获取内容的需求。
    note: 这段特别适合接到你前面那些“租赁店”“跑秋叶原”的资料页后面，说明那不是零散个体经验，而是有商业基础设施托着的。
    focus: true
  - label: 它很快从软件租赁转向新品与中古 PC 销售，说明市场重心在迅速成形
    original: >-
      新品・中古パソコン販売に業態転換
      中古パソコン出張買取サービスを開始
    translation: >-
      年表里也能清楚看到，Sofmap 很快就从软件租赁转向新品和中古电脑销售，并继续补上上门回收等服务。这说明九十年代前后用户需求已经不只是“借内容”，而是围绕机器、软件、转卖和再流通形成了一整套成熟市场。
    note: 这块很适合补“内容消费背后其实有完整的二手流通体系”。
    focus: true
  - label: 从秋叶原到大宫、名古屋、京都等地的扩张，意味着购买路径开始全国化
    original: >-
      駅前ターミナル立地で総合大型店舗「ギガストア大宮」を開店
      以降、全国に展開
    translation: >-
      年表中最关键的阶段之一，是大宫这类大型站前综合店的出现，以及之后向全国扩张。它标志着“去秋叶原买”的模式没有消失，但同时也开始被更广域、更标准化的零售网络部分替代。
    note: 这段非常适合解释为什么九十年代后半内容购买既还保留秋叶原中心性，又逐渐能在别处完成。
  - label: 后来的通信销售与电商展开，也说明购买环境从跑店慢慢过渡到了远程下单
    original: >-
      インターネットによる通信販売サービスを開始
      eコマース店舗「ソフマップ・ドットコム」を開店
    translation: >-
      到后续开启网络通信销售、再到独立电商店上线，Sofmap 的历史又展示了另一条转变：原本必须跑实体店完成的购买行为，开始逐步被远程下单和线上渠道吸走。这条变化也和你整理的九十年代末到二十一世纪初传播环境转折是同步的。
    note: 这块适合给“黄金时代后段为什么消费方式会变”补上一条零售侧证据。
