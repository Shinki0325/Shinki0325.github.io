  - label: PC-9800 的历史起点，是 1982 年那台定价 29.8 万日元的 PC-9801
    original: >-
      そして、1982年10月13日。ついに発表されたPC-9800シリーズには、298,000円という価格が設定されました。当時としては破格です。この日から国民機と呼ばれる名機の歴史が始まるのです。
    translation: >-
      这篇通史给出的关键起点非常明确：1982 年 10 月 13 日，PC-9800 系列正式亮相，定价 29.8 万日元。在当时这是极具冲击力的价格点，也正是从这一天开始，后来会被称作“国民机”的那条历史真正起步。
    note: 这段很适合主稿里第一次正式落下“PC-98 作为时代底盘”的时间钉。
    focus: true
  - label: 它不是凭空变成国民机的，而是踩着 PC-8000 与软件生态一路长出来的
    original: >-
      そうしたハンディがありながらもPC-8000シリーズは1979年に発売へとこぎつけ、ベストセラーになりました。そしてパソコンが一般のコンシューマーに受け入れられるようになったのです。ゲームなどのエンターテインメントはもちろん業務用のソフトが豊富にそろったエコシステムが築き始められました。
    translation: >-
      文中也提醒我们，PC-9800 不是凭空诞生的“神机”。在它之前，PC-8000 系列已经先一步把个人电脑推向更广的消费者，并逐渐长出游戏和商用软件都能共存的生态；PC-9800 是在这条基础上进一步吃下日本市场的。
    note: 这块适合避免把 PC-98 神话成单点奇迹，而是把它放回硬件与软件生态连续发展里看。
  - label: 它之所以吃下日本市场，一个核心原因是“面向日本人使用”的设计和软件策略
    original: >-
      この日発表されたのはPC-9801。のちに無印98と呼ばれることになるパソコンです。インテルの8086互換16ビットプロセッサーとしてNECオリジナルのμPD8086を搭載した製品で、日本語を処理するというビジネス用途には欠かせないさまざまな工夫がちりばめられていました。
    translation: >-
      文章强调，最初的 PC-9801 从硬件到显示设计都围绕“如何更好地处理日语和商务用途”来做取舍。这种强烈的本土化方向，再加上后续对第三方软件生态的掌握，正是它能在日本坐上主流位置的重要原因。
    note: 这段很适合解释“为什么很多九十年代内容型游戏会生长在这块平台上”，因为它背后是整套本土计算环境。
    focus: true
  - label: 1990 年代之后，DOS/V 与 Windows 最终把这条独自演化的路线吞没了
    original: >-
      1990年に、日本アイ・ビー・エムは「IBM DOS J4.0/V」をリリースします。
      この製品の登場で、PC-9801 vs. DOS/V機という図式が現実的な解としてできあがり、寡占状態だったPC-9800シリーズにかげりが見えるまでになっていくのです。
      そして、1997年、15年間続いたPC-9800シリーズはPC/AT互換機としてのPC98-NXシリーズとして生まれ変わります。
    translation: >-
      但这条路线并没有无限延续。随着 1990 年 DOS/V 的出现、1990 年代前半 PC/AT 兼容机逐渐渗透，再到 Windows 成为统一平台，PC-9800 那种日本本土独自演化的优势开始失效。到了 1997 年，它最终转向 PC98-NX，也就是转入 PC/AT 兼容机框架。
    note: 这段很适合把“黄金时代赖以成立的硬件土壤”与“它为什么终究会退场”连成一条完整的历史线。
    focus: true
