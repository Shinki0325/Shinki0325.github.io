  - label: 在系列回忆里，录像机不是玩具，而是几乎关乎家庭文化生存的设备
    original: >-
      第３回は、私にとってまさに愛機と呼ぶにふさわしいビデオデッキのお話です。
    translation: >-
      这篇长回忆最适合补的一点，是录像机在当年并不是普通家电，而是能决定“哪些节目能留下来、哪些内容能反复看、哪些共同记忆能形成”的关键设备。作者甚至直接把它称作自己真正意义上的“爱机”。
    note: 这能把“视听设备”从背景道具抬回到文化经验的核心位置。
    focus: true
  - label: 当时一盘录像带和一场长片录制，都是真金白银堆出来的
    original: >-
      しかし当時のビデオテープは結構な値段です。
      せっかくバイトで稼いだお金も４～５千円単位で目減りしていきました。
    translation: >-
      文中最能让人立刻感到年代差的一点，是录像带本身就很贵。作者说，自己辛苦打工赚来的钱，会因为一盘盘磁带和一次次长片录制而以几千日元为单位快速消耗，这让“保存内容”在当时从来不是零成本动作。
    note: 这段特别适合补“为什么当年的录制选择会带着很强的取舍意识”。
    focus: true
  - label: 没有录像机时，很多地方观众连本来就稀缺的节目接触机会都会直接断掉
    original: >-
      こうして購入からわずか一年あまりで我が家から「ビデオのある生活」が失われることになってしまいました。
    translation: >-
      作者回忆机器故障后，家里很快就失去了“有录像机的生活”。在今天看这像是设备坏了，但放回当时，它实际上意味着节目保存、错时观看、朋友间共享录制内容这些能力会一起被切断。
    note: 这块很适合和地方动画页联动，说明资源不平等不仅发生在电视台层面，也发生在家庭设备层面。
    focus: true
  - label: 录像机选择甚至会影响朋友之间能不能交换内容
    original: >-
      なんにしても、うちもVHSにしとかなお前のテープ貸してもらえんやないか。
    translation: >-
      文章里一个非常生动的细节是，朋友家最终改买 VHS，理由之一竟然是“这样才能和你借录像带”。这说明媒介格式本身也会重塑社交圈和内容流通路径，设备标准直接影响谁能和谁共享资源。
    note: 这一段很适合补“媒介基础设施如何进入同好关系和传播链”。
