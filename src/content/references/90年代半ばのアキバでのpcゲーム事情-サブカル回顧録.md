  - label: 90 年代中期是“游戏还在到处流通”的最后高峰
    original: >-
      唐突に語ることになるけど、90年代半ばの秋葉原は恐ろしくゲーム全盛期だったと言ってもいいと思う。Windows時代直前でプレステやサターンが出て間もない頃。スーファミが後期に入り、メガドラ、PCエンジンは末期とはいえまだまだゲームは市場にはかなりの数のソフトが流通していた。
    translation: >-
      作者开门见山地说，九十年代中期的秋叶原几乎可以直接称作“游戏全盛期”。那时离 Windows 时代全面到来还差一点，PS 和土星刚出不久，SFC 已进入后期，MD 和 PCE 虽然接近末期，但市面上依旧有极其可观的软件在流通。
    note: 这段适合给主稿定调，说明为什么九十年代中期会被许多人回忆成“最后什么都买得到的时刻”。
    focus: true
  - label: Sofmap 的中古楼层就是 PC 游戏搜索起点
    original: >-
      この時代の秋葉原のゲーム事情だがこの時代の秋葉原でのパソコンゲームの中心はやはりソフマップであろう。ソフマップに行ってまずはチェックすることからスタートすることになる。ここはフロアごとに中古ゲームソフトが分けられており、１階が確か新品、４階と５階が確かパソコンゲームソフトと言う感じだったと思う。そこには末期ながらPC9801系からX68000、FM-TOWNS用のゲームから、MSX、8801系等ところせましと並んでいたものだ。
    translation: >-
      在作者的记忆里，那一时期秋叶原 PC 游戏的中心显然就是 Sofmap。逛街的起点通常也是先去 Sofmap 检查库存；整栋楼按楼层分中古软件，到了四楼、五楼基本就是 PC 游戏区，PC-9801、X68000、FM-TOWNS、MSX、8801 等一整排摆得密密麻麻。
    note: 这块能把“去秋叶原买游戏”具体化成非常真实的店铺空间。
    focus: true
  - label: 98 末期反而是最容易低价入坑的时候
    original: >-
      ブライ上巻を買ったことがあるが確か500円で売られていて、こんな感じで名作ゲームが３桁台というのもザラだったので、多分9801でゲームをするにはこの90年代半ばは一番安価で出来た時代だったと思う。またエロゲーに関しては少しピークが遅くちょうどこの95年前後あたりが全盛期だったと思う。98のゲームは90年代中期にはエロゲーまみれになっていて、無茶苦茶な数が出ていて中古ゲームショップはそりゃあエロゲーが死ぬほど売ってました。
    translation: >-
      作者举例说，像《ブライ》上卷这种作品当时甚至 500 日元就能买到，很多名作落到三位数价格都不稀奇。所以如果只是想玩 PC-9801 游戏，九十年代中期反而可能是成本最低的入坑时代。与此同时，98 末期的成人游戏产量却又极高，中古店里几乎铺天盖地都是这类作品。
    note: 这段很适合支撑“98 末期的成人游戏不是边角料，而是压倒性库存的一部分”。
    focus: true
  - label: 2000 年前后，Windows 化让这套流通现场迅速消失
    original: >-
      この状況に変化が生じてきたのは2000年ぐらい。Windows時代になってパソコンが一気に進化していき、普及台数も極端に上がっていったが同時に古い独自パソコンの在庫が少しずつであるが減っていくのである。やはりいちばん大きかったのはソフマップ５号店が中古Windowsパソコンに舵切りし、同時にパソコンゲームを中古専門で扱っていたソフマップもWindows中古ゲーム中心に舵切りをしてしまったことだろう。
    translation: >-
      转折点大约出现在 2000 年前后。随着 Windows 时代到来，PC 迅速升级并大量普及，旧式国产机的库存也开始一点点被挤掉。作者认为最致命的一步，是 Sofmap 5 号店转向中古 Windows 机器，而原本主打 PC 游戏中古的软件楼层也一起改成以 Windows 游戏为中心，这等于宣布旧秩序开始瓦解。
    note: 这段很适合放在主稿里作为“黄金时代为何会突然像退潮一样消失”的实体零售版解释。
    focus: true
