《泡沫冬景》是我最近游玩的作品，网络上的评论我也大致看了下，这是一部推出之后便备受好评的作品，不过在不少人眼中，这部作品的缺点还是非常明显，基本可以归结为时代与人物联系不紧密，剧情白开水，剧情割裂/不合理，诸如此类。

我个人认为以片冈的风格来看，他对于大场景的设计确实比较无力，在剧情上不少地方也比较欠缺张力，人物维度，冲突抉择的设计上感觉都略显平淡，加上本作中日合拍的大力宣传，对于不少人来说确实会有实际与预期不符的感觉。不过这改变不了我对片冈的喜爱，生活少有新岛夕的狗血，难见虚渊玄的幻灭，更多的是片冈笔下平淡的真实，没有轰轰烈烈跌宕起伏的恋情，但就像我们一样可以为一点小小的感动而欣喜；抱怨命运的不公，却依然要继续选择面对。

接下来我将就本作具体细节陈述我的个人感想。

# 时代感

作为《平凡的世界》的读者，我在一开始也抱着比较大的期望，遗憾的是与路遥笔下将人物发展融入社会变革不同，本作只是断续的掠影，如一开始中日物价的悬殊对比，远渡重洋出卖人身的偷渡劳工，改革开放与恢复高考，电话中因产业转移和技术革命而即将失业的技工父亲，但这些时代的脉动几乎并没能与人物的发展相交织，这点确实让我有些失望。本作对于时代（泡沫）的塑造确实乏善可陈，更多的只是通过一些符号来呈现，所述时代离现在也相对久远，我相信对于大部分游玩者来说还是比较有距离感的，而且不仅是时间的距离，也是空间的距离。但是我认为这些描写并没有失真，仍然在一定程度上很好向我们展示了属于那个时代的一些侧面，想要在一位日本作家笔下尽量还原出上世纪七八十年代中国的生态，想必制作组一定还是下了很大功夫，抑或是制作组中有亲历者？

就我个人经历而言，我生活的地方和日本有着千丝万缕的联系，大街上偶遇的出租司机说不定正要向你饶有兴致地讲述日本的谋生经历；你的老师在课上讲起了操着一口方言在日本遇到的笑谈；大街上随处可见的，是日语培训，日本留学中介。从小耳濡目染着长辈日本打拼史的我，在游玩这部作品时感受到了似曾相识的熟悉感。

我的大伯在恢复高考之后一连参加了三次高考才考取了师专，之后是我的父亲参加高考，但也就是在那一年，我的祖父因病去世了，这场重大变故改变了我的家族，父亲高考落第后没能再考而是选择了能尽快就业的中专，为了偿还债务，大伯辞掉了教师的工作借了钱远赴日本，也就是那一年，我的表姐出生了，她的名字叫樱。

如果时代是一场戏，我相信我们都是剧中人。就像故事中的佐仓说的那样“在这个时代，大家每个人都有自己的故事”。

# 关于男主

父亲既不选择亡妻也不选择爱子，幼小的景萧然虽然埋怨着父亲从未为自己庆祝的生日，但也懂事的选择沉默。但父亲却在那个圣诞夜打破了静默，“或许两个都选，也可以吧”，临别的期望让童年对于父亲憧憬再次被点燃，萧然努力学习，想要考上北大。父亲想要他“成为出色的大人”，但在这之后更重要的期望却是“找到梦想和目标”。但他的渴望却因为高考前得知的父亲死讯而断线。

以重新开始为名，他来到了日本边打工边备考。但是如果萧然真的想考取大学，那么两年前他就应当选择复读，实际上他心底对于父亲认同的渴望，对于目标梦想的寻觅仍然深埋在心底，未曾随着父亲的逝去而消散，所谓“考大学”“体验工商”正是他这一不自觉欲望的体现，不过是一个幌子，尽管不知道目标梦想为何物，但他仍然想延续当年父亲的期望，他迷茫过，但与樱井姐妹的邂逅让他找回了人生的热量，再次放弃学业看起来与他当初的目标矛盾，却恰恰说明他在追寻中找到了真正的目标和梦想。

我觉得景萧然这个角色还是挺不错的，虽然迷茫，但仍然踏实的前进着，不过我感觉对于他的塑造仍然有所欠缺，角色性格需要通过压力下的抉择来揭示，但给予男主的压力总让我感觉没什么实感（迟钝直男kana），或者显得有些突兀，反倒有种压力都在女主身上的感觉，最开始看到景萧然陪江小墨对峙到后面栞走投无路时，我还想后面是不是会有男主在黑道与女主之间的艰难抉择，看来是我想多了，这么一看本作在角色塑造上确实有所失衡，不过，普通一点，说不定，也好吧。

（话外题：关于大家一直吐槽的男主日语学习问题，我觉得其实也没有那么不正常。其一，当时没有像现在这样丰富的日语教学资源，如果没有合适教材的话，单纯想要靠自学成才还是比较困难的，不过有趣的是，《标日》正是在故事主要发生的1988年出版的。其二，就我个人了解，出国多年不会外语这种事也是很有可能的，受限于语言，一般华人在海外都会抱团（就像故事中景萧然去找江小墨），也会有各种各样的华人聚集区，再者从事一些纯苦力活基本上也不需要什么日语交流，人毕竟都有惰性，可以理解，加上男主看上去就很自闭，跟樱井姐妹的交流也很有限，学的慢也正常，不过这主观能动性属实不行）

# 关于运气

在时代、命运的浪潮中，个人的力量实在是太过微不足道。

就如同西西弗斯的神话一般，每当我们积累起一些东西，命运就会无情的将其摧毁。

假如没有妹妹突如其来的心脏病，樱井一家可以快乐的生活，和其他家庭别无二致。

假如没有父亲的被迫害和死亡，景萧然大概率可以考入大学，成为一名受人敬仰的出色的大人。

假如没有对妻子一时的忽视和那意外的车祸，江小墨的生活理想也不会迎来终结。

只是...运气不好罢了...

这是故事中反复提到的一句话，这是他们对生活的妥协，生活并不因莫名的苦难而暂停，命运不因我们的嘶吼而改写，我们还要继续存在下去，或许不同于《西西弗神话》与《平凡的世界》中英勇迈向苦难的不屑，但这就是我们普通人能够做到的回应，我们并没有做错什么，只是运气不好罢了，但无论是樱井姐妹，，抑或是江小墨父女，他们都在继续坚强的生活着，他们给予我们的，是在生活的贫乏本质面前迎面而上的勇气。

# 后记

说实话不太会写感想，但还是努力写下来了，希望以后能写好点。我心目中这肯定是值得一试的作品，在演出效果上也很吸引我，希望国产galgame也能越来越好。
