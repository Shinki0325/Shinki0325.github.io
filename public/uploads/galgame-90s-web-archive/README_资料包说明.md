## 3. 当前研究主轴

这批资料当前支撑的主线，不是“证明黄油比动画更伟大”，而是：

- 90年代日本动画的新作传播存在明显地域格差与时差；
- 相比之下，PC 平台美少女游戏更容易形成全国范围内的共时性体验；
- 再叠加 PC-98 / Windows 过渡期的媒介条件、成人向题材空间与分支叙事优势，才构成了所谓“黄金年代”的特殊历史窗口。
